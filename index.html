    🍳 Eating Python for Breakfast

    Welcome to Eating Python for Breakfast!

    Learning to code shouldn't feel like swallowing an entire textbook whole. This site takes the massive, industry-standard 30 Days of Python curriculum by Asabeneh and chops it up into bite-sized, high-yield computing nybbles.

    A nybble is exactly half a byte (4 bits)—a small, fundamental unit of data. True to that first-principles mindset, every single day of this challenge is broken down into ultra-digestible concepts, interactive questions, and deep architectural explanations that you can expand at your own pace.

    🍽️ What's on the Menu?

    • First-Principles Breakdowns: No glossing over the details. We explain why things work under the hood (like why Python's execution model differs from strict, compiled languages like Go).
    • Active Recall Format: Every concept is served hidden behind a collapsible dropdown. Read the question, test your brain, and click to reveal the breakdown.
    • Side-by-Side Context: Where applicable, Python concepts are contrasted with other ecosystems to give you a broader mental model of computer science.

    🧭 How to Consume This Guide

    1. Pick a Plate: Head over to the sidebar and select a daily breakdown (starting with Day 1 Overview).
    2. Guess Before You Click: Treat the dropdowns like flashcards! Try to answer the question or explain the concept to yourself before expanding it.
    3. Write Code: Reading is only half the meal. Make sure to open up VS Code, spin up a Python shell, and run the code snippets yourself.

    Let's dig in! 🚀
